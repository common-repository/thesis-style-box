== Description ==

Thesis Users can Make Styled Note box, Alert box, Help box, Tip Box, Important Box in your posts using classes.If you are not a Thesis user checkout our Style Box Plugin for the Same.

== Installation ==

To install the plugin follow these steps :

1. Download the thesis-style-box.zip file to your local machine.
2. Unzip the file 
3. Upload "thesis-style box" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

That's it.
